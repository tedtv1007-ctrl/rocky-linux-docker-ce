# 企業級私有鏡像倉庫 Harbor 部署指南 (Rocky Linux 10)

本指南說明如何在 Rocky Linux 10 上使用 Docker Compose 部署 Harbor，作為企業內部的私有容器鏡像倉庫 (Private Registry)。

## 1. 為什麼選擇 Harbor？
- **基於角色存取控制 (RBAC)**：精細管理使用者對專案的存取權限。
- **安全性掃描 (Trivy)**：自動掃描鏡像中的漏洞 (CVEs)。
- **鏡像複製 (Replication)**：支援在多個 Harbor 實例或與公有雲間同步鏡像。
- **Helm Chart 託管**：可作為 Helm Chart Repository 使用。

## 2. 前置準備
- 已安裝 Docker 與 Docker Compose (參考 [Docker 安裝指南](./docker-install.md))。
- 建議規格：4 CPU / 8GB RAM / 100GB Disk 以上。
- 準備一個域名 (如 `harbor.example.com`)。

## 3. 安裝步驟

### 下載離線安裝包 (推薦)
```bash
export HARBOR_VERSION=v2.11.0
wget https://github.com/goharbor/harbor/releases/download/${HARBOR_VERSION}/harbor-offline-installer-${HARBOR_VERSION}.tgz
tar xvzf harbor-offline-installer-${HARBOR_VERSION}.tgz
cd harbor
```

### 配置 harbor.yml
複製範例檔並進行編輯：
```bash
cp harbor.yml.tmpl harbor.yml
vi harbor.yml
```

**核心修改項：**
- `hostname`: 設定為你的域名 (例如 `harbor.company.com`)。
- `http`: 預設埠 80。
- `https`: (強烈建議) 配置證書路徑，或在測試環境暫時註解掉 HTTPS 部分（改用 HTTP）。
- `harbor_admin_password`: 設定管理員密碼。

### 執行安裝腳本
```bash
# 同時安裝 Trivy 安全掃描與 Helm Chart 倉庫功能
sudo ./install.sh --with-trivy --with-chartmuseum
```

## 4. 登入與使用
1. 瀏覽器訪問 `http://your-ip`。
2. 使用帳號 `admin` 與你設定的密碼登入。

### 推送鏡像測試
```bash
# 登入 Harbor
docker login your-harbor-domain

# 標記並推送
docker tag my-app:latest your-harbor-domain/library/my-app:v1
docker push your-harbor-domain/library/my-app:v1
```

## 5. K8S 存取設定 (ImagePullSecrets)
若 K8S 需從私有 Harbor 拉取鏡像，需建立 Secret：
```bash
kubectl create secret docker-registry harbor-registry-secret \
  --docker-server=your-harbor-domain \
  --docker-username=admin \
  --docker-password=your-password \
  --namespace=default
```
