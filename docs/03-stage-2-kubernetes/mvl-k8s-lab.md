# 最小可行性 K8S 實驗室 (Rocky Linux 10)

本文件提供一個在單機 Rocky Linux 10 環境下快速啟動最小化 Kubernetes 叢集 (KinD) 的方案，用於本地開發與測試。

## 方案選擇：KinD (Kubernetes in Docker)
在企業環境中，使用 KinD 可以極速啟動符合標準的 K8S 叢集，且完全依賴你已安裝的 Docker 環境。

### 1. 安裝 KinD 二進位檔
```bash
# 下載並安裝 KinD (amd64)
curl -Lo ./kind https://kind.sigs.k8s.io/dl/v0.24.0/kind-linux-amd64
chmod +x ./kind
sudo mv ./kind /usr/local/bin/kind
```

### 2. 建立叢集設定檔 (kind-config.yaml)
為了模擬真實環境，我們建立一個包含一個 Control-plane 與兩個 Worker 節點的叢集：

```yaml
kind: Cluster
apiVersion: kind.x-k8s.io/v1alpha4
nodes:
- role: control-plane
- role: worker
- role: worker
```

### 3. 啟動叢集
```bash
kind create cluster --config kind-config.yaml --name enterprise-lab
```

### 4. 驗證
```bash
kubectl cluster-info --context kind-enterprise-lab
kubectl get nodes
```
